# Index - 指标函数

| 函数名             | 说明                                                                                                                    |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------- |
| `sar(n, s, m)`     | 求抛物转向指标，`n` 为计算周期，`s` 为步长，`m` 为极值                                                                  |
| `sarturn(n, s, m)` | 求抛物转向点，`n` 为计算周期，`s` 为步长，`m` 为极值，若发生向上转向则返回 `1`，若发生向下转向则返回 `-1`，否则返回 `0` |
