# String - 字符串函数

| 函数名                    | 说明                                    |
| ------------------------- | --------------------------------------- |
| `datestr(x)`              | 以格式 `YY-MM-DD` 格式化日期整数 YYMMDD |
| `lowercase(x)`            | 转换为小写字符串                        |
| `uppercase(x)`            | 转换为大写字符串                        |
| `numtostr(x)`             | 转换数值为字符串                        |
| `numtostrn(x, n)`         | 转换数值为字符串，精确到小数点后 `n` 位 |
| `strtonum`(x)             | 解析字符串为数值                        |
| `concat(x1, x2, x3, ...)` | 连接字符串                              |
